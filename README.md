### [미래내일 일경험] 팀 프로젝트

이 저장소는 [해오름 인포텍] 교육 과정에서 진행한 4인 팀 프로젝트의 결과물을 포크(Fork)한 것입니다.

원본 저장소: https://github.com/team-ballon/team-ballon-java-backend

프로젝트 역할: 저는 팀원으로서 WBS, ERD 설계 토론에 참여했으며, Figma를 사용하여 '회원가입-로그인-장바구니-결제'로 이어지는 업무 플로우 차트(User Flow Diagram) 작성의 기반을 마련했습니다. 제가 그린 초안을 바탕으로 팀원들과 논리 흐름을 검토하고 최종본을 완성할 수 있었습니다.장바구니 기능 구현을 위한 SQL 쿼리 학습 및 일부 카테고리의 웹 크롤링 스크립트 작성을 담당했습니다. 

회고: 이 프로젝트를 통해 Git-flow 기반의 협업(PR, Merge) 프로세스와 복잡한 DB 설계를 경험할 수 있었습니다.
